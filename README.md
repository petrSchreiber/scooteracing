# ScooteRacing

Retro stylized Scooter Racing game done in 7 nights for an entertainment at author's wedding.

The state of game after 7 nights is reflected in commit 89d0a0298c46bfbcc7b3b1f1970e8e8b93c08ef4. 

## Requirements
- GPU with OpenGL support
- 2x Xbox One controllers
- thinBasic 1.11.7.0 or newer

## Known issues
- the framerate oscilates between fixed 30 and 60 FPS
- under specific circumstances, you can take a shortcut via grass
